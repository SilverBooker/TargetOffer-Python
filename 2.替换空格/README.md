## 替换空格
### 题目描述
请实现一个函数，将一个字符串中的每个空格替换成“%20”。
例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy。
### 解法
从右到左扫描字符串替换
* 统计空格数，字符串长度增加 空格数 * 2。
* 维持两个指针p1, p2。
* p1指向原字符串长度末尾， p2指向新字符串长度末尾。
* 当p1遇到空格时， p2 向前移动并替换字符为 '0' '2' '%', 若此时p1 == p2, 结束替换， 否则p1, p2都向前移动。
* 把p1指针指向的字符拷贝到p2指针指向的字符。

所有字符都只移动一次，时间复杂度为O(n) 。
